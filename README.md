## Devil May Cry HD Collection Mod Loader (DMCHDLoader)

Serves as a mod loader for the DMC Archipelago randomizers also functions as a replacement loader for both Crimson and DDMK.

If other mods for the HD Collection also use "dinput8.dll" to load, please open an issue so I can add support for them.

Currently supported mods are as follows:

Devil May Cry 1:
- DDMK (Eva)
- Archipelago Randomizer

Devil May Cry 2:
- DDMK (Lucia)

Devil May Cry 3:
- DDMK (Mary)
- Crimson
- Archipelago Randomizer
